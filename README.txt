LaSNAKE

By Lucy Blatherwick (lb13g16), May 2018

An implementation of the game Snake for the LaFortuna.


Other Code:
  - lcd.c uses Steve Gun's code (adapted by Klaus-Peter Zauner)


Features:
  - Snake grows as it eats food.
  - Score increases with length of snake.

Missing Features/Issues:
  - Levels - The snake does not yet get faster as score increases.
  - The screen does not refresh at an acceptable rate.
